# Agora Blockchain

Agora Blockchain is a decentralized voting platform built on blockchain technology. It enables secure, transparent, and fully anonymous elections through zero-knowledge proofs and the Semaphore library. Agora abstracts blockchain complexities, providing a seamless user experience for admins and voters alike.

## Features

- User registration and management
- Election creation and management
- Multiple voting algorithms (e.g., approval voting, quadratic voting)
- Fully anonymous voting using zero-knowledge proofs
- Real-time election result subscriptions
- Intuitive and user-friendly interface
- Secure and transparent vote counting
- Accessibility for non-technical users

We welcome contributions from the community! If you'd like to contribute to Agora Blockchain, please follow these steps:
